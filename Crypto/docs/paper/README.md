# Paper Kit

This directory packages the current repository into a paper-ready framing.

Recommended positioning:
- research system / validation framework
- not alpha-claim paper
- not live trading paper

Recommended title direction:
- governance-aware strategy validation
- LLM-assisted crypto strategy research
- multi-asset and regime-robust validation

Files:
- `venue_strategy.md`: realistic submission targets and positioning
- `abstract.md`: draft abstract and contributions
- `outline.md`: paper structure
- `experiment_plan.md`: concrete experiment matrix and tables
- `deepresearch_prompts.md`: prompts to send to a separate research workflow

Current strongest claim:
- The repository implements a reproducible AI-assisted strategy validation pipeline that combines proposal generation, mutation lineage, multi-asset testing, market regime testing, overfitting controls, governance gates, artifact persistence, and approval/registry logic.

Current weakest claim:
- The repository does not yet prove superior economic performance versus strong baselines in a publication-grade benchmark.
